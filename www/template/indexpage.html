But this is not your average website on Apache (although Apache is used for proxying). I wrote the code that serves these webpages myself, in a obscure little language called C. The webserver is threaded and is select() based for portability and performance.

The pages are sent from memory, and preloaded when the webserver starts. Also, a MySQL backend holds the content, like this page.
